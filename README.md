# 我的百草园

我知道自己才疏学浅，但也总希望能够有一个地方记录自己所学，所想，所悟。经常研究各种各样的技术，但每每细想起来，总觉得自己研究的并不深刻透彻，如同雾里看花，似懂非懂。出入江湖时，信心满怀，希望有朝能够在技术领域有一席之地，名震一方。然而毕业已有五年，越发觉得技术变革日新月异，我们总在跟随，无法超越。

未来呢...
